# Snipe-IT Product Overview

Snipe-IT is an open-source IT asset management system built on Laravel 11. It helps organizations track:

- Hardware assets (laptops, monitors, etc.)
- Software licenses
- Accessories and consumables
- Components
- Asset depreciation
- Maintenance records
- User assignments and checkout history

## Key Features

- Asset checkout/checkin workflow with acceptance signatures
- Custom fields and fieldsets per asset model
- QR code and barcode generation for asset labels
- LDAP/SAML/OAuth authentication support
- REST API with Passport authentication
- Multi-company support with data isolation
- Audit logging and reporting
- Email notifications (Slack, Teams, Google Chat integrations)
- CSV import/export capabilities
- Localization (50+ languages via Crowdin)
